# Stage 1 - Priority Inbox

This project implements a Priority Inbox Notification System.

## Features

* Displays top 10 unread notifications
* Priority-based sorting
* Recency-based ordering
* REST API implementation
* React frontend UI

## Tech Stack

* React.js
* Node.js
* Express.js

## Run Project

### Backend

```bash
cd backend
node server.js
```

### Frontend

```bash
cd frontend
npm run dev
```
